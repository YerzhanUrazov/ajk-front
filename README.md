# Заявки — Админ-панель

Статический React-интерфейс в стиле enterprise admin по макету «Заявки».

## Запуск

```bash
npm install
npm run dev
```

Откройте в браузере адрес, который покажет Vite (обычно http://localhost:5173).

## Сборка

```bash
npm run build
npm run preview   # просмотр собранной версии
```

## Структура

- **Шапка** — заголовок «Заявки»
- **Боковая панель** — поле поиска и навигация по разделам (Новые, В работе, Исполненные, Отмененные) с счётчиками
- **Основная область** — заголовок, кнопки «Обновить» и «Выйти», таблица заявок с колонками: Номер, Заявитель, Адрес, Статус, Дата, Действия (кнопка «Открыть»)

Цвета, отступы и стиль кнопок соответствуют макету. Backend не используется — все данные статичные.
